## Formatting Rules

### Business Name
- Always use: `Southwest Resume Services`
- Never abbreviate to "SRS" or "Southwest Resumes" in NAP listings
- Legal entity name: `Southwest Resume Services, LLC`

### Address
- Use `N` not `North`
- Use `Blvd` not `Boulevard`
- Use `#2016` for suite (not `Suite 2016` or `Ste 2016`)
- Use `AZ` not `Arizona`
- Always include ZIP code `85224`

### Phone Number
| Context | Format | Example |
|---------|--------|---------|
| Website display | US parentheses | (480) 374-3418 |
| tel: href links | E.164 with dashes | +1-480-374-3418 |
| Schema.org markup | E.164 with dashes | +1-480-374-3418 |
| Plain text for citations | US parentheses | (480) 374-3418 |
