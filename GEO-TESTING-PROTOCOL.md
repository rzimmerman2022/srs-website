## Executive Summary

### What is GEO Testing?

**Generative Engine Optimization (GEO)** is the practice of optimizing your brand's visibility in AI-generated responses. Unlike traditional SEO (which focuses on search engine rankings), GEO measures whether AI models like ChatGPT, Claude, Perplexity, and Google AI Overviews mention your brand when users ask relevant questions.

### Why GEO Matters for SRS

As more people use AI chatbots for research and recommendations, traditional search is declining. If someone asks ChatGPT "best resume writer in Phoenix" and your business doesn't appear in the response, you've lost a potential customer‚Äîeven if you rank #1 on Google.

**GEO testing helps you:**
- **Track brand awareness** in AI systems over time
- **Identify gaps** in your online presence
- **Measure ROI** of content marketing efforts
- **Discover** what AI systems know (or don't know) about your unique methodologies
- **Stay ahead** of competitors in the AI-driven discovery landscape

### How to Use This Protocol

1. **Weekly Quick Tests** (15 minutes): Run 5 core branded queries
2. **Monthly Comprehensive Tests** (60 minutes): Run all 30 queries across all platforms
3. **Quarterly Competitor Analysis** (90 minutes): Track competitor mentions and compare positioning
4. **Track scores** in the provided spreadsheet template
5. **Take action** based on findings (see [Improvement Actions](#improvement-actions))

## What is GEO Testing?

### The Four Key Metrics

When you submit a test query to an AI platform, measure these four dimensions:

#### 1. **Mentioned?** (Yes/No)
- **Question:** Does the AI mention Southwest Resume Services by name?
- **Scoring:** 1 point for Yes, 0 points for No

#### 2. **Position** (1st-7th cited)
- **Question:** If mentioned, what position is SRS in the response?
- **Scoring:**
  - 1st citation = 7 points
  - 2nd citation = 6 points
  - 3rd citation = 5 points
  - 4th citation = 4 points
  - 5th citation = 3 points
  - 6th citation = 2 points
  - 7th+ citation = 1 point
  - Not mentioned = 0 points

#### 3. **Accuracy** (0-3 points)
- **Question:** Is the information about SRS accurate?
- **Scoring:**
  - 0 = Mentioned but with incorrect information
  - 1 = Vague/generic mention (e.g., "Southwest Resume Services is a resume company")
  - 2 = Accurate but basic (e.g., mentions location and service type correctly)
  - 3 = Detailed and accurate (mentions unique methodologies, correct pricing, specific frameworks)

#### 4. **Framework Citation** (Bonus 0-2 points)
- **Question:** Does the AI mention any SRS-specific frameworks or concepts?
- **Frameworks to watch for:**
  - Client Truth Principle
  - Research Authority Index (RAI)
  - Truth Gap
  - Five-Level Enhancement Scale
  - Truth Bridge Protocol
  - Document Risk Index (DRI)
  - Ten-Dimensional Quality Framework
  - Three Pillars of Ethical Enhancement
  - Dual Baseline Scoring
  - Gap Narrative Mastery
- **Scoring:**
  - 0 = No framework mentioned
  - 1 = One framework mentioned
  - 2 = Two or more frameworks mentioned

### Maximum Score Per Query

**Total Possible:** 12 points per query
- Mentioned: 1 point
- Position: 7 points (if 1st)
- Accuracy: 3 points
- Framework Citation: 2 points (bonus)

### Target Scores by Query Type

**Branded Queries (SRS-specific):**
- **Target:** 8-12 points
- **Why:** AI should know your own brand and methodologies

**Local Queries (Phoenix/Arizona):**
- **Target:** 5-8 points
- **Why:** Local visibility is competitive; appearing at all is success

**Industry Queries (Non-branded):**
- **Target:** 3-5 points
- **Why:** These are highly competitive; any mention is strong performance

**Comparison Queries:**
- **Target:** 4-6 points
- **Why:** Being included in "best of" lists demonstrates authority

## Scoring Methodology

### How to Score Each Test

#### Step 1: Submit the Test Prompt
Copy the exact test prompt from the [Test Query Library](#test-query-library) and paste it into the AI platform (ChatGPT, Claude, Perplexity, etc.).

#### Step 2: Read the Full Response
Review the entire AI response. Don't just scan for your brand name‚Äîread carefully.

#### Step 3: Score Each Dimension

**Dimension 1: Mentioned? (0-1 points)**
- Search the response for "Southwest Resume Services" or "SRS"
- If mentioned: **1 point**
- If not mentioned: **0 points**

**Dimension 2: Position (0-7 points)**
- Count where SRS appears in the list of recommendations
- Example: "Here are top resume services: 1) TopResume, 2) Southwest Resume Services..." = 2nd position = 6 points
- If not mentioned: **0 points**
- If mentioned 1st: **7 points**
- If mentioned 2nd: **6 points**
- If mentioned 3rd: **5 points**
- If mentioned 4th: **4 points**
- If mentioned 5th: **3 points**
- If mentioned 6th: **2 points**
- If mentioned 7th or later: **1 point**

**Dimension 3: Accuracy (0-3 points)**
- **0 points:** Information is incorrect (wrong location, wrong services, fabricated details)
  - Example: "Southwest Resume Services is based in California" (WRONG - you're in Arizona)
- **1 point:** Vague/generic mention with no specific details
  - Example: "Southwest Resume Services is a resume company"
- **2 points:** Accurate but basic (correct location, service type, general description)
  - Example: "Southwest Resume Services is a Phoenix-based resume writing service"
- **3 points:** Detailed and accurate (mentions unique methodologies, correct pricing range, specific frameworks)
  - Example: "Southwest Resume Services in Phoenix uses the Client Truth Principle and Research Authority Index to create research-backed resumes. Pricing ranges from $150-$500+."

**Dimension 4: Framework Citation (0-2 bonus points)**
- Count how many SRS-specific frameworks are mentioned:
  - Client Truth Principle
  - Research Authority Index (RAI)
  - Truth Gap
  - Five-Level Enhancement Scale
  - Truth Bridge Protocol
  - Document Risk Index (DRI)
  - Ten-Dimensional Quality Framework
  - Dual Baseline Scoring
- **0 points:** No frameworks mentioned
- **1 point:** One framework mentioned
- **2 points:** Two or more frameworks mentioned

#### Step 4: Calculate Total Score
Add all four dimensions together.

**Example Calculation:**
- Mentioned? YES = 1 point
- Position: 3rd = 5 points
- Accuracy: Detailed and accurate = 3 points
- Framework Citation: Mentioned "Client Truth Principle" and "RAI" = 2 points
- **TOTAL: 11 points out of 12 possible**

#### Step 5: Record in Tracking Sheet
Log the score in your tracking spreadsheet (template provided in [Tracking Template](#tracking-template)).

### Testing Best Practices

1. **Use Incognito/Private Browsing:** Prevent personalization from affecting results
2. **Test from Consistent Location:** Use VPN set to Phoenix, AZ if testing remotely
3. **Use Exact Prompts:** Copy-paste from Test Query Library (no variations)
4. **Test at Consistent Time:** Same day/time each period (AI models update frequently)
5. **Document Model Versions:** Note which ChatGPT/Claude version you're testing
6. **Save Raw Responses:** Keep screenshots or text copies for later reference
7. **Test New Platforms:** As new AI assistants launch, add them to rotation

## Improvement Actions

### Scenario 1: Low Branded Query Scores (Below 8/12)

**Problem:** AI systems don't know enough about your own brand.

**Root Causes:**
